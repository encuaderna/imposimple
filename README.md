# ImpoSimple

**App para imposicion de documentos PDF** — pensada para quienes encuadernan y necesitan convertir cualquier PDF en un cuadernillo listo para imprimir y coser.

## Que hace esta app

- Convierte archivos PDF en cuadernillos listos para encuadernar
- Permite definir el orden de paginas para impresion en duplex
- Incluye opciones de accesibilidad: fuente para dislexia, alto contraste, texto escalable
- Disenada para principiantes, sin necesidad de conocimientos tecnicos

## Como usarla

Haz clic en el siguiente enlace y empieza directamente en el navegador, sin instalar nada:

**Abrir app:** https://print-masked-flow-pro.base44.app

## Para quien es

Para cualquier persona que encuaderne artesanalmente y quiera preparar sus documentos de forma simple y rapida.
